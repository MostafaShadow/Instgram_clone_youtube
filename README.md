## Demo
https://instgram-clone-mern.herokuapp.com/


## Install dependencies

```
  Server:
  npm install
```

```
  Client:
  bash
  cd client/
  npm install
```

Start the server

```bash
  npm start
```
Start the Client

```bash
  cd client
  npm start
```
